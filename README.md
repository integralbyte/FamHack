# Famhack - Academic Family Hackathon

<img width="1512" height="861" alt="image" src="https://github.com/user-attachments/assets/03537090-a7e0-4ad5-8c5c-3873bf58e5eb" />

A fun, modern website for **FamHack**, a family themed hackathon with live challenges, team building, demos, food, prizes, and a pre-hackathon CTF.

* One day academic family hackathon
* Pre-challenge CTF warm-up
* Easter eggs for prizes
* Live brief, side quests, and scavenger hunt
* Build, solve, present, or strategise
* Clean event info, schedule, FAQ, and registration flow
